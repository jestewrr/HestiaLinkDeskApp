# Quick Start - HestiaLink Desktop App

## Run Your Desktop Application

### Visual Studio:
1. Open `HestiaIT13Final.sln`
2. Press **F5** (or click the green play button)
3. Select "Windows Machine" as the target

### Command Line:
```powershell
cd "c:\Users\Jester\source\repos\HestiaIT13Final"
dotnet run --project HestiaIT13Final.csproj
```
