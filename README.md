# HestiaLink Desk App

State-of-the-art Hotel Management System built with **.NET MAUI Blazor Hybrid**.

## Prerequisites

To open and run this solution successfully, you **MUST** have the following installed:

1.  **Visual Studio 2022 (v17.12 or later)**
    *   *Note: .NET 9 is currently in Preview for some VS versions. Ensure you have a compatible version.*
2.  **.NET 9.0 SDK**
3.  **.NET MAUI Workload**
    *   Install via Visual Studio Installer > Workloads > **.NET Multi-platform App UI development**.

## Setup Instructions

1.  Clone the repository.
2.  Open **`HestiaIT13Final.sln`** (Do NOT use other solution files if present).
3.  Wait for dependencies to restore.
4.  Set `HestiaIT13Final` as the Startup Project.
5.  Select **Windows Machine** as the target.
6.  Run the application (F5).

## Troubleshooting

### "Project not loaded" or SDK Errors
*   Ensure you have .NET 9.0 installed (`dotnet --version`).
*   Check that `global.json` matches your installed SDK version.
*   Update Visual Studio to the latest release.
